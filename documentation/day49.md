# Malpractice & Integrity Detection – Zecpath AI

## Objective
To ensure fair and trustworthy AI-based interviews.

## Key Features
* **Real-time monitoring**: Browser hooks detect tab switching and focus changes on the fly.
* **Multi-signal detection**: Integrates voice inputs, eye gaze tracks, and tab events.
* **Risk scoring**: Translates events into a scale subtraction score from 100.
* **Warning system**: Triggers real-time alerts warning candidates to focus on the screen.

## Advantages
* **Prevents cheating**: Discourages tab switching or looking up concepts online.
* **Improves hiring quality**: Identifies high-risk candidates.
* **Ensures fairness**: Holds all candidates to identical evaluation standards.

## Limitations
* False positive risks (e.g. looking away due to thinking or tab loss due to system popups).
* Depends on device camera permissions and browser layout limits.

---

## Ethical Considerations
* **Requires user consent**: Trackers will not initialize unless candidate explicitly selects agreement tags.
* **No intrusive monitoring**: The application never reads candidate hard drives or system process logs.
* **Privacy-first design**: Tracks only numerical event counters rather than screen streams or raw video.

## Future Improvements
* AI-based pattern anomaly detection models.
* Voice biometrics identification checking for speaker overrides.
* Deep behavioral pattern classification.
