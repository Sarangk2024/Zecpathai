# Zecpath AI Ethics Framework

## Objective
To ensure the HR AI system operates in a fair, transparent, and responsible manner while protecting candidate rights and data.

## Core Ethical Principles

### Fairness
* No discrimination based on gender, age, religion, caste, or location.
* Scoring based only on job-relevant signals.

### Transparency
* All AI decisions must be explainable.
* Recruiters can view scoring breakdown.

### Accountability
* Every AI decision is traceable.
* Logs maintained for audit purposes.

### Privacy
* Candidate data is securely stored.
* Sensitive demographic data is masked before evaluation.

---

## Consent Requirements

### Candidate Consent Policy
Before starting the AI interview, the candidate must be informed and explicitly agree to the following policies:
* **AI Evaluation**: Candidates must be informed that an AI system will evaluate their responses.
* **Data Processing**: Candidates must accept that their data will be stored and processed.
* **Recording**: Candidates must accept that voice/audio may be recorded during the sessions.

### Data Handling Rules

| Data Type | Action |
| --- | --- |
| **Resume** | Stored securely with restricted access control |
| **Audio** | Converted to text via speech-to-text; raw audio is optional |
| **Personal Info** | Masked before scoring/evaluation stages |
| **Logs** | Stored securely for audit purposes |
