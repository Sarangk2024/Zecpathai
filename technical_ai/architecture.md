# Zecpath Technical Interview AI – System Design

## Objective
To conduct role-based, adaptive technical interviews with intelligent difficulty progression.

## Core Components
1. **Role & Skill Mapper**
   - Maps job roles to required skills (e.g., MERN → JS, React, Node, DB)
2. **Question Engine**
   - Generates technical questions based on:
     - Role
     - Experience level
     - Difficulty
3. **Difficulty Engine**
   - Adjusts complexity dynamically based on candidate responses
4. **Answer Evaluation Engine**
   - Checks correctness
   - Evaluates logic & depth
5. **Follow-Up Engine**
   - Probes deeper based on answers
6. **Technical Scoring Engine**
   - Accuracy
   - Depth
   - Optimization thinking
7. **Report Generator**
   - Technical strengths
   - Weak areas
   - Final recommendation

## System Flow
```
Start → Role Mapping → Question Generation → Answer Evaluation → Difficulty Adjustment → Score → Report
```
